=======
Sharedy
=======
Sharedy is a demo application to show how to create a Facebook like Share
Stream

 * `Demo:` http://www.sharedy.com
 
 Install
 =======
 Download the Google App Engine SDK `here
 <http://code.google.com/appengine/downloads.html>` and clone this repository.
 
 You can then start the dev server by running::
 
     $ path/to/google_appengine/dev_appserver.py -p 8000 /path/to/sharedy/.


That's it. Enjoy.
